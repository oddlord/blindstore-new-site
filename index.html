Blindstore

{{ site.description }}. Provides an online lookup service which returns the correct answer without the server ever knowing what the question was.

Centralized

Unlike Distributed Hash Table, not prone to Sybil attack, and the data are not inherently public, since server is free to restrict access with authentication and authorization

Private

Protocol guarantees that the server can't know what the client's query was, and that the client doesn't know anything except the record she inquired about.

Here be dragons

Blindstore is early work in progress and is made using experimental cryptographic software. It can be used for research purposes only.
